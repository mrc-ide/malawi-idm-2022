Infectious Disease Modelling with a focus on Malaria

Welcome to the Malawi-Liverpool-Welcome and Imperial College London workshop Infectious Disease Modelling with a focus on Malaria.

This workshop will be taught using an interface built using an R package called shiny using the R programming language. You will be using the interface to explore and understand the basics of infectious disease modelling.
